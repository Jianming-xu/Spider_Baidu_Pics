在此基础上，笔者：

1.添加了多线程操作以提升下载速度

2.删去了获取推荐功能

3.添加了运行时间的输出

使用方法：首先在name.txt中分行输入想要爬取的图片关键字，如‘爱尔兰’‘郭德纲’‘猫’等，然后运行.py文件，输入每类图片下载的张数并运行。


实测同时爬取6个词条 每个词条40张图片耗时250秒左右。
